# ProgressIQ

ProgressIQ is a productivity and goal-decomposition web application that helps users transform ambitious long-term goals into structured milestones, actionable tasks, and measurable progress metrics.

## Features

* Create and manage goals
* Add milestones to goals
* Track progress visually
* Organize long-term objectives into actionable steps
* Persistent storage (coming soon)

## Tech Stack

* HTML
* CSS
* JavaScript
